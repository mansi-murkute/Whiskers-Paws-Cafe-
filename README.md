# Whiskers & Paws Cafe 🐱☕

A pixel-art cat cafe simulation game built with Pygame. Feed cats, order drinks & food, collect a cute bag, and read classic cat poems!

## Features
- 5 unique cats with expressions (happy/sad, blinking, tail wag)
- Feed cats multiple treats (8 options)
- Order from 9 menu items with a shopping cart
- Collect order with a pixel-art bag and serving message
- Cat poems: Macavity, The Naming of Cats, etc.
- Pixel-art graphics with particles and sparkles

## How to Run
1. Install Python 3.8+
2. Install Pygame: `pip install pygame`
3. Run `python cat_cafe.py`

## Controls
- Click buttons and cats with your mouse
- Scroll poems with mouse wheel

Enjoy the purrs!
